PURPOSE

CLINE - plots a 3D curve (x,y,z) encoded with scalar color data (c)

SYNOPSIS

function h_out = cline(x,y,z,c,cmap,crange)

DESCRIPTION 

 CLINE - plots a 3D curve (x,y,z) encoded with scalar color data (c)
 using the specified colormap (default=jet);
