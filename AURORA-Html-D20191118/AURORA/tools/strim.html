strim

PURPOSE

STRIM(S) strip the trailing and leading blanks of a string.

SYNOPSIS

function s1 = strim(s)

DESCRIPTION 

STRIM(S) strip the trailing and leading blanks of a string.
         STRIM(S)is an extension of the MATLAB function DEBLANK(S)
         and removes both leading and trailing blanks from the string S.
